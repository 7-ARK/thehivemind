# TheHiveMind Frontend

Next.js dashboard for the TheHiveMind MVP. It renders the command panel, run timeline, agent workspace, task graph, metrics, memory, and final output.

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:3000`.

Set `NEXT_PUBLIC_API_BASE_URL` if your backend is not running at `http://127.0.0.1:8000`.
