# TheHiveMind Frontend

Vite React dashboard for TheHiveMind orchestration and usage telemetry.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3001`.

The frontend expects the FastAPI backend at `http://127.0.0.1:8000`.
Set `VITE_API_BASE_URL` if your backend is running somewhere else.
